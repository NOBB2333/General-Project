# 经营报表中心与扩展设计

## 1. 修订说明

本文件沿用原编号，当前改为三期经营管理区的报表与后续扩展设计。

## 2. 报表中心定位

报表中心在三期里的定位是：

- 跨项目检索入口
- 明细筛查入口
- 经营预警明细入口

它不是自由报表平台，也不替代项目经营页。

## 3. 当前设计原则

- 固定报表优先
- 数据口径统一
- 结果能跳回项目经营页
- 报表中心样式服从数据表达，不做花哨外壳

## 4. 后续扩展方向

- 预算类分析扩展
- 合同与回款分析扩展
- 经营预警对象扩展
- 与外部经营数据的同步边界扩展

## 5. 当前结论

三期的扩展重点不是搭平台，而是把经营总览、项目经营、报表中心这三者的关系真正稳定下来。
