# 组织与数据范围

组织范围决定用户在租户内能看到哪些业务数据。

## 组织归属

用户可以归属一个或多个组织。角色授权中的数据范围会基于这些组织计算可访问组织集合。

## 常见范围

- 全部：可看租户内全部组织数据。
- 本组织及下级：可看自己组织和子组织的数据。
- 本组织：只看自己所在组织。
- 仅本人：只看与自己直接相关的数据。
- 自定义：由授权配置指定组织集合。

## 项目模块补充规则

项目天然存在跨组织协作，因此项目可见性除了组织范围，还会考虑项目关系：

- 项目经理可见。
- 项目发起人可见。
- 项目成员可见。
- 历史成员在允许历史只读时可见。

这能避免 PM 或成员因为项目组织不在自己组织范围内而完全看不到参与项目。
