# Introduction

General Admin is an enterprise management platform for platform governance, project delivery, business analytics, and deployment operations.

This English section is intentionally minimal for now. The primary documentation is maintained in Chinese under the root locale.

## Scope

- Platform governance: tenants, users, roles, menus, organization scopes, and audit logs.
- Project delivery: projects, tasks, milestones, risks, issues, members, and documents.
- Business analytics: contracts, budgets, procurement, receivables, profit, and warnings.
- Operations: local development, NUC deployment, database migration, and troubleshooting.
